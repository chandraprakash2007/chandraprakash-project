# 💰 FinPulse — Daily Expense Analytics Dashboard

A modern, single-page expense tracker built with React.

## Features
- ✅ Add income & expense transactions
- ✅ Real-time summary (Total Income, Expenses, Net Balance)
- ✅ Expense breakdown by category with animated progress bars
- ✅ Transaction history with delete support
- ✅ Clean dark dashboard UI — no external libraries

## Live Demo
🔗 Deployed via Vercel

## Tech Stack
- React 18
- Vite
- Pure CSS (no UI libraries)

## Run Locally
```bash
npm install
npm run dev
```
